![React](https://img.shields.io/badge/React-18-61DAFB?logo=react&logoColor=black)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-5-646CFF?logo=vite&logoColor=white)

A single-page portfolio site built with React, TypeScript, and Vite. Serves as a companion to my resume, covering experience, projects, and background in more depth than a resume format allows.

**Live site:** [adityakm100.github.io/aditya-kanginaya-portfolio](https://adityakm100.github.io/aditya-kanginaya-portfolio/)

## Stack

- React + TypeScript
- Vite
- Plain CSS (no framework)

## Pages

- **Home** — intro
- **Projects** — featured and full project list, including research, applied ML, and engineering work
- **Experience** — internship and research roles
- **Skills** — technical skills by category
- **Contact** — links and contact info

## Getting started

Install dependencies:

```powershell
npm install
```

Run the dev server:

```powershell
npm run dev
```

## Build

```powershell
npm run build
npm run preview
```

## Deployment

Deployed via GitHub Pages from the `dist` build output.
